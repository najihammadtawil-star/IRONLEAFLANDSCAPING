IRONLEAF MEDIA FOLDER SYSTEM

Use this folder to organize temporary GPT-created visuals now and real job photos/videos later.

Main folders:
- 00-unsorted-dropbox: Put new images/videos here when you are not sure where they belong.
- 01-home-hero: Strongest brand image or video for the first screen.
- 02-home-transformations: Best before/after or high-impact homepage visuals.
- 03-project-gallery: Gallery/project section media.
- 04-about-company: Crew, process, equipment, or brand visuals.
- 05-service-area: Maps or area-related visuals.

Service folders:
- services/design-installs
- services/sod-installation-turf-replacement
- services/beds-planting
- services/cleanup-trim
- services/drainage-grading
- services/tree-services
- services/exterior-cleaning
- services/outdoor-upgrades-christmas-lights

Before/after folders:
- before-after/landscape-design
- before-after/sod
- before-after/cleanup-refresh
- before-after/tree-services
- before-after/exterior-cleaning

Naming tips:
- Use simple lowercase names with hyphens.
- Good: katy-front-yard-after-01.jpg
- Good: sod-install-before-01.jpg and sod-install-after-01.jpg
- Good: exterior-cleaning-driveway-01.mp4
- Avoid spaces and long random names when possible.

Swap workflow:
1. Put temporary GPT-created visuals into the right folders.
2. Later, replace them with real photos/videos using similar filenames.
3. If the filename stays the same, the website can keep working without code changes.
4. If filenames change, tell Codex which file should replace which section.
